# Parser
Тестовое задание на должность Junior Python Developer

## Описание
Программа парсит новостные порталы и получает текст
статьи в отформатированном виде.
Правила форматирования:
- ширина строки не более 80 символов
- абзацы и заголовки отбиваются пустой строкой
- ссылки вставляются в квадратных скоблка

Статья сохраняется во вложенной дирректории.
Например:
<i> https://lenta.ru/news/2021/03/28/sur_suec/ => 
[CUR_DIR]\lenta.ru\news\2021\03\28\sur_suec\index.txt </i>

## Запуск программы
Программа запускается из консоли с указанием двух параметров:
- URL
- тэг, по которому ищется основная статья
